❓ 진행자 대시보드 사용방법
🚀 시작하기
  • · 방 코드 입력 후 팀장 수 설정 → 🏪 경매 시작
  • · 팀장들에게 방 코드 공유 (🔑 코드 보기 버튼)
  • · 📋 팀원 일괄 추가로 엑셀/CSV 또는 직접 입력으로 팀원 등록
🏪 경매 진행
  • · 🔔 경매 시작 버튼으로 현재 선수 경매 시작 (10초 카운트다운)
  • · ✋ 입찰: 팀장 선택 + 포인트 입력 → 입찰 (팀장들도 직접 가능)
  • · 🏆 낙찰 확정: 카운트다운 종료 후 최고 입찰자에게 낙찰
  • · ⏭ 유찰: 입찰자 없으면 유찰 → 나중에 재경매
  • · ↩ 취소: 마지막 입찰 취소 / 🔄 낙찰 취소: 낙찰 롤백
  • · 🔀 랜덤 셔플: 경매 순서 무작위 섞기
  • · ✏️ 편집 버튼 (큐 카드 하단): 팀원 정보 수정/삭제
👑 팀장 현황 패널 (우측)
  • · 팀장 카드 클릭 → 해당 팀 상세보기 (팀원 정보 전체)
  • · 포인트 빨간색 = 80점 미만 주의
  • · 💰 포인트 지급/회수: 특정 팀장 또는 전체 일괄 조정
🏆 경매 종료
  • · 모든 선수 처리 후 자동으로 결과 화면 전환
  • · 🏆 경매 완료 버튼으로 수동 종료도 가능
  • · 📊 결과 요약 보기: 캡처용 전체 결과 화면
  • · 📥 전체 로그 CSV: 모든 입찰 기록 다운로드
💬 실시간 채팅
  • · 우하단 💬 버튼으로 채팅창 열기/닫기
  • · 👥 참여자 탭에서 접속 중인 인원 목록 확인
  • · 🚫 금지 버튼으로 특정 유저 30분 채팅 금지 (진행자 전용)
